# AI 직원 만들기 🧑‍💼 (ai-agent-maker)

> 코딩을 몰라도 됩니다. **"무엇이 불편한지"만 말하세요.** 나머지는 AI가 질문해서 만들어줍니다.

이 스킬은 비개발자가 AI 에이전트(=AI 직원)를 쉽게 만들 수 있게 도와주는 **코치**입니다.
"~만들어줘"라고 명령하는 대신 "~하고 싶다"고 목표만 말하면, 코치가 알아서 질문하고
선택지를 주고, 실제 구현까지 데려갑니다.

## 무엇을 해주나요?

- 당신이 **목표만** 말하면 → 코치가 **필요한 것만 하나씩 질문**합니다.
- 어려운 용어는 안 씁니다. "외부 서비스 연결", "업무지침"처럼 **쉬운 말**로 설명합니다.
- 갈림길에선 **추천안을 먼저** 알려주고, 당신은 **고르기만** 하면 됩니다.
- 위험한 일(삭제·결제 등)은 항상 **먼저 물어봅니다**.

| 외부 서비스 연결 도우미 | `service-connector` | 구글시트·쇼핑몰 관리자 등 외부 서비스 연결(MCP) 안내 |

| 배포·공유 코치 | `deploy-helper` | 만든 AI 직원을 실제로 쓰거나 팀원·고객과 공유 |

## 설치 방법

### Claude Code
1. 이 저장소를 내려받습니다.
   ```bash
   git clone https://github.com/<your-id>/ai-project-coach.git
   ```
2. `ai-project-coach/.claude/skills/ai-agent-maker` 폴더를 통째로
   `~/.claude/skills/` 안에 복사합니다.
   (또는 이 저장소를 Claude Code로 열면 자동으로 인식됩니다.)
3. 실행:
   ```
   /ai-agent-maker
   ```
   또는 그냥 "뭔가 자동화하고 싶은데 어디서부터 할지 모르겠어"라고 말하면 됩니다.

### Codex
1. 위와 같이 저장소를 내려받습니다.
2. Codex로 이 저장소 폴더를 엽니다. 루트의 `AGENTS.md`를 자동으로 읽습니다.
3. "요즘 이게 제일 번거로워" 하고 목표를 말하면 코치가 시작됩니다.
