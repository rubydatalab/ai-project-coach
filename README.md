# AI 직원 만들기 🧑‍💼

> 코딩을 몰라도 됩니다. **"무엇이 불편한지"만 말하세요.** 나머지는 AI가 질문해서 만들어줍니다.

비개발자 셀러가 **대화만으로** AI 직원(=Agent)·자동화를 만들 수 있게 돕는 **Codex 코치 묶음**입니다.
"~만들어줘"라고 명령하는 대신 "~하고 싶다"고 목표만 말하면, 코치가 알아서 질문하고
선택지를 주고, 실제 구현까지 데려갑니다.

## 무엇을 해주나요?

- 당신이 **목표만** 말하면 → 코치가 **필요한 것만 하나씩 질문**합니다.
- 어려운 용어는 안 씁니다. "외부 서비스 연결", "업무지침"처럼 **쉬운 말**로 설명합니다.
- 갈림길에선 **추천안을 먼저** 알려주고, 당신은 **고르기만** 하면 됩니다.
- 위험한 일(삭제·결제 등)은 항상 **먼저 물어봅니다**.

## 설치 (Codex)

**① Codex 설치 + 로그인** (딱 한 번)

Windows PowerShell에서:
```powershell
irm https://chatgpt.com/codex/install.ps1 | iex
```
설치 후 PowerShell을 닫았다 다시 열고, `codex`를 실행해 **Sign in with ChatGPT**로 로그인합니다.
(Node가 이미 있으면 `npm install -g @openai/codex`로 깔아도 됩니다.)

**② 코치 플러그인 설치** (딱 한 번)

```
codex plugin marketplace add rubydatalab/ai-project-coach
codex plugin add ai-employee-coach@ai-project-coach
```

**③ 부르기**

그냥 한국어로 목표를 말하면 됩니다 — "요즘 이게 제일 번거로워, 자동으로 하고 싶어".
무엇부터 할지 모르겠으면 **AI 직원 만들기**(`ai-agent-maker`) 코치가 상황에 맞는 전문 코치로 안내합니다.

## 전체 코치 목록 (11종)

| 묶음 | 코치 이름 | slug | 언제 쓰나 |
|---|---|---|---|
| 🏁 시작 | **AI 직원 만들기** (마스터) | `ai-agent-maker` | 뭘 해야 할지 모를 때. 여기서 시작해 아래 코치로 안내 |
| 🛠 준비 | 환경 준비 도우미 | `setup-helper` | 브라우저 연결·외부 서비스 연결 등 필요한 것만 알아서 설치 |
| 🏢 사업 | 자동화 아이디어 코치 | `automation-finder` | "뭘 자동화하면 좋을지 모르겠어" — 하루 일과에서 대상 발굴 |
| | 사업 지식 노트 | `knowledge-keeper` | 내 사업 지식·들은 강의(녹취·메모)를 정리해 쌓아둠 |
| | 업무지침 만들기 | `instruction-writer` | AI 직원 매뉴얼(말투·금지사항·우선순위) 작성 |
| | 외부 서비스 연결 | `service-connector` | 구글시트·쇼핑몰 등 데이터 연결(MCP) 안내 |
| | 브라우저로 일 시키기 | `browser-operator` | 로그인하는 웹사이트에서 반복 작업을 AI에게 시킴(학습모드) |
| | 일하는 순서 만들기 | `workflow-builder` | "먼저 ~하고 그다음 ~한다" 일하는 순서 설계 |
| 🙋 개인 | 개인 비서 | `personal-assistant` | 스케줄·할일 관리, 챙겨야 할 일 정리 |
| 🔧 사용·문제 | 내 PC에서 쓰기 | `run-helper` | 만든 AI 직원을 내 컴퓨터에서 매일 켜서 쓰는 법 |
| | 문제 해결 코치 | `trouble-helper` | AI 직원이 이상하게 동작할 때 원인 좁히기 |

> **처음이라면** "뭔가 자동화하고 싶은데 어디서부터 할지 모르겠어"라고 말하세요. 마스터 코치가 안내합니다.
